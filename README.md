## 🍅 What is this?

**Ketchup Skills** is a collection of high-performance, AI-native skills designed to supercharge your coding assistant (Claude, Cursor, etc.). 

Built on the **Model Context Protocol (MCP)**, these skills allow your AI to interact with your codebase, generate multimedia content, and maintain a consistent brand voice across your engineering artifacts.

## 🚀 Available Skills

### 🎬 `ketchup-changelog`
**Turn git commits into cinematic narrative stories.**

Stop writing boring changelogs. This skill analyzes your git history and transforms it into:
- **Narrative Blog Posts**: engaging stories about *why* changes matter.
- **Video Scripts**: ready-to-render scripts for 60s explainer videos.
- **Social Content**: tweets and LinkedIn posts that get engagement.

```bash
# Install this skill
npx skills add gitketchup/ketchup-skills --skill ketchup-changelog
```

## 🛠️ Usage

Once installed, simply ask your AI assistant:

> "Generate a changelog story for the features we shipped this week."

> "Create a video script for the new Auth v2 release."

> "Analyze the impact of my recent commits."

## 📦 Installation

This repository is fully compatible with [Skills.sh](https://skills.sh).

# Add the Ketchup Skills registry
npx skills add gitketchup/ketchup-skills
