## 🍳 How to Cook (Usage)

All skills are **MCP-compatible**, meaning they work natively with Claude Desktop, Cursor, and any other MCP client.

### 1. Install via Skills.sh
```bash
npx skills add gitketchup/ketchup-skills --skill ketchup-screenshot
```

### 2. Bring Your Own Key (BYOK)
We give you the utensils; you buy the groceries.
- **LLM Skills**: Require `ANTHROPIC_API_KEY` or `OPENAI_API_KEY`.
- **Voice Skills**: Require `ELEVENLABS_API_KEY`.

### 3. Let it Simmer
Ask your agent:
> "Take a smart screenshot of the 'Settings' page using the ketchup-screenshot skill."
