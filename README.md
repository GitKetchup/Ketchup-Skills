# Ketchup Skills

AI agent skills for code analysis and changelog generation.

## Installation

**Coming soon:**
```bash
npx skills add gitketchup/ketchup-skills --skill ketchup-changelog
```

**Development mode (available now):**
```bash
# Install MCP server from narrative-engine
cd ../ketchup-webapp/narrative-engine
pip install -e .
ketchup-mcp
```

## Available Skills

- [ketchup-changelog](skills/ketchup-changelog/) - Transform git commits into narrative stories, analyze codebase health, and generate video recaps.

## Features

- 📊 Code complexity analysis (multi-language)
- 🔒 Security vulnerability scanning
- 🤖 AI-generated code detection
- 📈 Development momentum scoring
- 👥 Contributor skills mapping
- 📝 Automated changelog generation
